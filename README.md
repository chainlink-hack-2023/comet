# Run CCIP Swap

First install Dependencies:

```bash
yarn
```

## Run

Start CCIP Swap on `http://localhost:3000`

```bash
yarn start
```

## Build

Build the project. The static files will be generated in the `build` folder.

```bash
yarn build
```

## Unit testing

```bash
yarn test
```
